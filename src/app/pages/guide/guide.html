Portfolio Guide

How to Navigate This Portfolio

by Jerome Mari Parucha

01 What is this?

This is my personal portfolio, designed to feel like a Steam profile. Instead of a traditional resume page, it's structured around the things I've built, the skills I've developed, and the experience I've accumulated — all navigable from the top bar.

02 Profile

The landing page. It shows who I am, my work experience, featured projects, certifications, a skills preview, and contact information. Think of it as the "player card" of the portfolio.

03 Library

A browsable collection of every project I've built. Use the search bar to find something specific, the filter panel to narrow by type, tech stack, or year, and the sort controls to reorder results. Click any project to open its detail page.

Project detail page

Each project has a full-width banner, a stats bar (last updated, tech count, status), and tabs for Overview, Screenshots, and Tech Stack. The sidebar has direct links to the live demo and source code.

04 Skills

My full tech stack, grouped by category — Languages, Frameworks, Developer Tools, DevOps, and Automation. Each skill is displayed as a badge.
